# PR Tracker

> PR 리뷰 코멘트, 하나도 빠뜨리지 않고 반영하세요.

리뷰 코멘트가 10개 이상 달리면 골라서 반영하다가 꼭 하나를 놓칩니다.

**PR 링크만 넣으면** 리뷰 코멘트가 체크리스트로 정리되고, 반영할 때마다 진행률이 올라갑니다.

<!-- 배포 링크: https://your-username.github.io/pr-tracker -->

## ✨ 주요 기능

### 리뷰 체크리스트 & 진행률

- 리뷰 코멘트를 한 장씩 카드로 -> 리뷰어, 작성 시각, 파일 경로·라인 표시
- 반영 완료 체크 시 진행률 갱신, 100% 달성 시 완료 알림
- 체크 상태는 PR별로 저장되어 다시 열어도 이어짐

### 리뷰 분류 & 정렬

- "LGTM 👍" 같은 답변 불필요 코멘트는 자동으로 진행률에서 제외 (수동 되돌리기 가능)
- 파일 순 / 시간 순 / AI 중요도 순 / 우선순위(P1~P5) 순 정렬
- 전체 · 미완료 · 완료 · 리뷰 불필요 필터

### 스레드 & 답글 연동

- 코멘트 아래 스레드 전체 표시, 내 답글은 "나" 배지로 구분
- **여기서 쓴 답글이 GitHub PR에 바로 등록** (repo 권한 토큰 필요)

### 코드 패널

- 코멘트가 달린 diff, PR 변경 파일 목록·내용을 사이드 패널로 확인
- 리뷰 당시 코드 ↔ 수정된 현재 코드 비교로 반영 여부 검증
- vscode.dev 링크로 해당 파일·라인 바로 열기

### 학습 노트

- 리뷰 코멘트·답글·내 메모를 분석해 공부할 주제 자동 추출
- 주제별 추천 자료 링크(MDN, 스타일 가이드 등) + 학습 체크리스트

### PR 아카이브

- 열어본 PR을 폴더별로 자동 아카이빙, 반영 진행 상태 한눈에
- 아카이브 전체를 합산한 통합 학습 노트

## 🚀 시작하기

1. **+ 새 PR 등록** 클릭
2. 저장할 폴더 이름 지정 (예: `우테코`, `회사`)
3. GitHub PR 링크 붙여넣기 → 완료

처음이라면 **데모 데이터로 체험하기**로 전체 흐름을 먼저 볼 수 있습니다.

### GitHub Access Token

공개 저장소는 토큰 없이 사용 가능합니다. 아래 경우에 토큰을 등록하세요.

| 필요한 경우                     | 권한      |
| ------------------------------- | --------- |
| 비공개 저장소 PR 조회           | `repo`    |
| 답글을 PR에 등록                | `repo`    |
| 시간당 API 요청 제한(60회) 해제 | 아무 토큰 |

발급: GitHub Settings → Developer settings → Personal access tokens
(fine-grained token으로 필요한 저장소만 허용하는 것을 권장합니다)
